# SVG Workspace Milestones

SVG Workspace follows this roadmap:

1. Preview
2. Optimization Engine
3. Icon Workspace
4. Sprite Builder
5. Continue Working

## Current Focus

The current focus is the first public Preview and then the Optimization Engine.

Destination-specific workflows such as React and Vue are intentionally postponed.
